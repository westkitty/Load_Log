# Potential Features for Load Log

1.  **Tagging System**: Add custom tags to events for more flexible categorization beyond the presets.
2.  **Duration Tracking**: Optional start/end time logging for analysis of session length.
3.  **Wellness Check-ins**: Periodic prompts to log general mood/health even without specific intimacy events.
4.  **Menstrual Cycle Overlay**: Visualize cycle phases on the timeline/charts for context.
5.  **Medication Logs**: Track PrEP or other relevant medication intake alongside events.
6.  **Partner Nicknames**: Assign color codes or avatars to partner aliases for quick visual identification.
7.  **Location Heatmap**: Privacy-preserving visualization of event locations (if location logging is used).
8.  **Templates**: "Quick Add" buttons for frequent event types (e.g., "Regular Solo").
9.  **Streak Counting**: Optional motivational counters (e.g., "Days since last..." or "Streak of...").
10. **Journal Prompts**: Optional reflective questions after logging specific moods.
11. **Dark/Light Mode Toggle**: Manual override for system theme preference.
12. **Biometric Unlock**: Use device FaceID/TouchID (via WebAuthn) as a shortcut to the passphrase (still requires passphrase as fallback/encryption root).
13. **Data Aging**: Auto-delete events older than X months (for data minimization).

15. **Import from Other Apps**: Scripts to parse data from generic CSVs.
16. **Hidden 'Duress' Mode**: A secondary passphrase that unlocks a dummy/empty account.
17. **Calendar Integration (One-way)**: Export events to device calendar (sanitized titles only).
18. **Custom Rating Scales**: Allow 1-10 or emoji-only scales instead of just 1-5 stars.
19. **Year in Review**: A special yearly summary card generated locally.
20. **Random Memories**: "On this day" feature to resurface past positive memories.
